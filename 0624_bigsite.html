03 HISTORY / CONTEXT

2009

インバウンド施策スタート

家庭薬業界として、一緒に動く。

龍角散・太田胃散など家庭薬各社が連携し、訪日外国人向けのインバウンド共同活動を開始。単独では難しかった施策を、業界団体として仕掛ける構造をつくった。

2009年 インバウンド共同活動

2009 インバウンド施策スタート
03 HISTORY / CONTEXT

2010

暢游日本・インバウンド冊子

訪日前に、正しい情報を届ける。

家庭薬共同企画として訪日ガイドブック『暢游日本』を刊行。大阪心斎橋・北海道狸小路などの店頭にPOPを設置し、訪日客への露出を強化した。

暢游日本 / インバウンド冊子

2010 暢游日本・店頭接点
03 HISTORY / CONTEXT

2014

「神薬」指定・爆買いの始まり

口コミが、商品の存在を変えた。

2014年9月、中国SNS・小紅書（RED）で「日本に行ったら買わなければならない12品」に選定。同年10月の免税制度スタートとともに百度検索が急増し、店頭での爆買いが始まった。

2014年 小紅書での紹介 / 爆買い

2014 「神薬」指定と爆買い
03 HISTORY / CONTEXT

2014〜

越境EC・デジタル展開

日本を離れても、龍角散と繋がる。

越境ECプラットフォーム（天猫・京東・小紅書）での販売を開始。中国語公式HP（香港サーバー）と公式SNS（微博・微信・小紅書）を立ち上げ、正規情報の発信体制を整えた。

越境EC / 中国語HP・SNS

2014〜 越境EC・公式SNS
03 HISTORY / CONTEXT

2017

中国現地広告展開

現地で、ブランドをつくる。

2017年より中国本土での広告展開を本格化。教師向けイベントは2020年以降定番化。現地の消費者に「龍角散を知ってもらう」投資を継続した。

2019

華潤三九との戦略提携

現地の力と、組む。

2020

コロナ禍を越えて

インバウンドが止まっても、止まらなかった。

コロナ禍で訪日客がゼロになる中、越境ECと中国向け一般貿易を強化。伊藤忠食品経由でのテスト販売を経て2021年には大河実業・華潤三九との代理店契約を締結し、販路を多様化させた。

04 MARKETING IMPLICATIONS

COMMON MISTAKE

越境・輸出から始めると、
ブランドは届きにくい。

知名度が十分に形成されないまま海外展開を試みると、現地で“探される理由”が弱くなる。

インバウンドで選ばれていない商品は、海外でも選ばれにくい。

RYUKAKUSAN MODEL

龍角散が実証した、
インバウンドから世界への順序。
